经典游戏
======


贪吃蛇、俄罗斯方块等简单而好玩的经典游戏,相信80后、90后并不陌生。它们堪称是风靡一时的手游始祖。看似简单的玩法,却经久耐玩。
那时候还没有琳琅满目的网络游戏，也没有安卓、苹果，更没有当下玩法多如牛毛的手游。

本章将用掌控板和Python编程,复刻经典黑白游戏、缅怀消逝的童年、致敬经典！
